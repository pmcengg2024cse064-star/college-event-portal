# Logo Files

## Instructions to Add PMIC Tech Logo

1. **Save the logo image** to this folder with the filename: `pmic-logo.png`

The logo should be:
- Filename: `pmic-logo.png`
- Format: PNG with transparent background
- Size: At least 200x200 pixels (recommended 500x500)
- Location: `public/images/pmic-logo.png`

Once you add the logo, it will automatically appear in the header at the top of the website.
